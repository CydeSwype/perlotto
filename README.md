# Perlotto - minimal google music wrapper

![Screenshot](./perlotto.png)

Perlotto is a minimal wrapper around Google Music service that turns it
in a decent player but does not add tons of crap on top. So, what's
inside?

* Multimedia keys
* Last.FM scrobbling
* No notifications

That's it.

## Development

```bash
$ npm install
$ npm start
```

## Packaging

```bash
$ npm run dist
```
